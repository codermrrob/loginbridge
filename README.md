# Demo Service for Enoki Backend for Canon Protocol login bridge

A demo service created as part of the proof of concept using decentralised storage and blockchain for Canon Protocol package management.

The service bridges Google and Twitch account logins with Enoki zkLogin service for zk wallets 

This code is not considered production ready.
